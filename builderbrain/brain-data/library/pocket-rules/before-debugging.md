# Before Debugging Rules

- Read the full error message — do not guess the root cause
- Reproduce the error in isolation before fixing
- Check logs and run logs for recent changes that may have caused the issue
- Search self-learning memory for known bug patterns before starting
- Identify the root cause, not just the symptom
- Fix one thing at a time — don't change multiple variables simultaneously
- Re-run the failing test after each fix attempt
- Save the lesson to bug-patterns.md after resolving
